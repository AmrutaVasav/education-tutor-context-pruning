# AI Education Tutor using Context Pruning

## Project Overview
This project builds a simple AI Education Tutor that answers student questions using a dataset. 
It uses Context Pruning to select only the most relevant information before generating an answer.

The goal is to help students in remote areas of India get educational assistance even with limited resources.

## Features
- Simple question answering system
- Uses context pruning to select relevant information
- Lightweight and easy to run
- Works with small datasets

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the project:

python app.py

Then ask a question.

Example:
What is photosynthesis?
